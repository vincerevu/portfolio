## Overview

Flazz is a local-first AI desktop coworker designed to keep user data on-device while still supporting long-term assistance and workflow automation.

## What I built

- Built the desktop app with TypeScript, Electron, and React.
- Developed prompt-based task execution flows for everyday productivity use cases.
- Added structured memory and reusable skill generation to make agent behavior more consistent over time.
- Worked on a modular architecture across renderer, main process, preload, and IPC layers.
- Supported external platform integrations through a unified tool-based workflow system.

## Why it matters

This project reflects my interest in practical AI systems that combine good product UX, local-first privacy, and reliable tool orchestration.
