## Overview

This project is a full-stack commerce platform with distinct applications for storefront, administration, and backend services.

## What I built

- Implemented authentication, product management, shopping cart, order processing, payment integration, and search and filtering.
- Used Spring Security, OAuth2, and JWT to support secure account and session flows.
- Added Redis, WebSocket, Cloudinary, and Swagger/OpenAPI for caching, real-time notifications, media handling, and API documentation.
- Built admin-side analytics and operational tooling for product and order management.

## Stack

Spring Boot, Next.js, React, TypeScript, PostgreSQL, Redis, WebSocket, and Cloudinary.
